Analyze the cryptocurrency market and provide trading recommendations following this workflow:

1. **Market Analysis (BTC)**: Analyze BTC market conditions with technical indicators
2. **Market Analysis (Altcoin)**: If analyzing altcoin, provide technical analysis
3. **News Analysis**: Analyze latest news and trends affecting the market
4. **Bear Research**: Build comprehensive case for SHORT position
5. **Bull Research**: Build comprehensive case for LONG position
6. **Neutral Research**: Build comprehensive case for HOLD/avoid position
7. **Research Manager**: Evaluate all arguments and create investment plan
8. **Trader Decision**: Make final trading decision with leverage, entry, SL, TP

For each step, provide detailed analysis following the agent guidelines in `.claude/agents/` folder.

The user will specify which cryptocurrency to analyze.
